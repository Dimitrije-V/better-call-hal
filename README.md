# better-call-hal


Letting and employment contract review tool powered by ChatGPT.


How to run locally:

1. clone repo
2. npm i
3. create a .env file
4. paste in  OPENAI_API_KEY=[a valid OpenAI API key]
5. npm run dev 
6. go to to localhost:3000
